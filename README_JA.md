# Timebox

固定された時間枠で作業を計画するための、シンプルなタイムボクシング Web アプリです。

アカウント登録は不要です。データはブラウザ内に保存されます。

[English version](README.md)

## デモ

https://isikoro1.github.io/timebox/

## コンセプト

Timebox は、1日の予定をカレンダー形式のタイムラインに配置し、作業を明確な時間枠の中で進めるためのツールです。

現在の UI では **15分単位** を編集の基本単位にしています。

- 1ブロック = 15分
- 空き時間をダブルクリックして予定を追加
- 予定をドラッグして時刻や日付を変更
- 予定の上下をドラッグして15分単位でリサイズ
- 今日の列に現在時刻ラインを表示

## 主な機能

- 日付ベースの予定管理
- 最大31日分の横スクロール表示
- カレンダーポップオーバーによる日付移動
- 和暦と日本の祝日ラベル表示
- 現在時刻ライン
- ラベル、メモ、URL を含む予定詳細編集
- 予定ブロックからの URL リンククリック
- JSON エクスポート / インポート
- タブを開いている間のアラーム通知
- タッチ操作でのスワイプ移動とピンチズーム

## 設計

### サーバーレス構成

- 認証なし
- データベースなし
- GitHub Pages 向けの静的エクスポート
- 予定データはブラウザの `localStorage` に保存

### 時間管理ロジック

- 時刻は内部的に0時からの分数として保存
- UI 上の編集は15分単位にスナップ
- ドラッグやリサイズでは、ポインター位置を分に変換して正規化
- 旧形式の曜日ベースデータは日付ベースの予定へ移行可能
