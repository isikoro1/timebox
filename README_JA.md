## 🧭 コンセプト

イーロン・マスクが実践している時間管理術として知られる「タイムボクシング」に着想を得て開発しました。

Notionカレンダーで実践する中で、高い集中状態を維持できることを体感し、  
より直感的で専用設計されたツールを作りたいと考え、本アプリを制作しました。

現在は **15分単位で時間を正規化** しています。

- 1枠 = 15分
- ドラッグ / リサイズは15分単位でスナップ
- 終わらない作業は後続枠へ再配置
- 現在時刻を可視化

時間を強制的に区切ることで、集中力の最大化を目指しています。

---

## ✨ 主な機能

- 週表示 / 3日表示の切り替え
- 空き枠ダブルクリックでイベント追加
- ドラッグで移動（時間・曜日）
- 上下ドラッグでリサイズ（15分単位）
- 現在時刻ライン表示
- イベント詳細編集（ラベル / description）
- URLリンク表示機能
- アラーム機能（タブが開いている前提で通知）

## 🏗 設計

### サーバーレス構成

- 認証なし
- データベースなし
- 静的サイトとしてエクスポート
- データはブラウザの `localStorage` に保存

GitHub Pagesで公開可能な構成としています。

---

### 時間管理ロジック

- 内部的には「分」で時間を保持
- UI上では15分単位でスナップ
- ドラッグ位置 → 分へ変換 → 正規化 → 状態更新

表示と内部状態を分離することで、ロジックを単純化しています。

---

### 開発運用

- GitHub Flow（feature / fix ブランチ運用）
- PR → squash merge
- main ブランチを常に安定状態で維持

実務に近い開発フローを意識しています。
